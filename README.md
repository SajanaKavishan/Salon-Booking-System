# Modern Salon Management System ✂️

A professional, full-stack mobile application designed to manage salon operations smoothly. Built with the modern MERN stack and React Native.

## 🚀 Features
- **Role-Based Access (RBAC):** Separate interfaces for Admin and Customers.
- **Smart Booking:** Real-time slot management for appointments.
- **In-store POS:** Efficient billing and invoice tracking for offline payments.
- **Staff Management:** Track stylist availability and schedules.
- **Service Catalog:** Full management of salon services (CRUD).

## 🛠️ Tech Stack
- **Frontend:** React Native
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT (JSON Web Tokens) & bcrypt

## 📂 Project Structure
This repository follows an industry-standard MVC pattern for the backend and a component-based architecture for the mobile app.
